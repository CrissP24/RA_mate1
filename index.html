Cómo Usar el Sistema

  1. Inicia Sesión: Usa tu cuenta para acceder al dashboard.
  2. Elige un Juego: Selecciona entre sumas, fracciones, geometría y más.
  3. Explora en VR: Usa un navegador compatible con WebXR (Chrome, Firefox).
  4. ¡D maticas! Aprende mientras te diviertes en entornos interactivos.

Requisitos para VR: Recomendamos dispositivos como Oculus Quest o cualquier visor compatible con WebXR. Asegúrate de usar un navegador actualizado.

Consejo: Configura un espacio seguro para la experiencia en realidad virtual y supervisa a los niños mientras juegan.
